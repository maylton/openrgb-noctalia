# Contributing

1. Create a branch from `main`.
2. Keep the canonical plugin id `maylton/openrgb-noctalia`.
3. Run `./scripts/validate.sh`.
4. Test against a live Noctalia v5 session and at least one OpenRGB controller.
5. Describe detected devices, OpenRGB modes and the Noctalia version in bug
   reports.

Do not commit personal palette files, OpenRGB profiles, device serial numbers,
runtime logs, or files from `$XDG_RUNTIME_DIR`.
