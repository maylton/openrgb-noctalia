# Changelog

## 0.1.1 — 2026-06-28

- Replaced the unsupported `string.trim` call with a Luau-compatible whitespace trim helper.
- Restored IPC color handling, Noctalia scheme inspection, state persistence and OpenRGB application.
- Added validation to prevent the unsupported helper from being reintroduced.

## 0.1.0 — 2026-06-27

- Initial standalone Noctalia v5 plugin.
- Native plugin settings and Control Center shortcut.
- Direct-mode OpenRGB synchronization without fixed device indices.
- Selective pastel-color correction.
- Album Aura palette integration.
- Nocky exact-palette bridge integration.
- Plugin-managed Noctalia primary-color bridge.
- Install, uninstall, validation, tests, translations, and GitHub Actions CI.
